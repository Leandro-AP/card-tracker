# Card Tracker

A standalone, desktop application for creating and managing virtual binders for trading cards.
Built with [Tauri](https://tauri.app/), [React](https://react.dev/), [TypeScript](https://www.typescriptlang.org/), [TailwindCSS](https://tailwindcss.com/), and [SQLite](https://www.sqlite.org/).

## Development

```bash
# Install dependencies
npm install

# Run in desktop dev mode
npm run tauri dev

# Create .exe in card-tracker/src-tauri/target/release
npm run tauri build
